#### **（1）编号原则**

1）用三位或三位以下的数字组成

需要标明回路的相别或某些主要特征时，可在数字标号的前面（或后面）增注文字符号。

2）按“等电位“的原则标注

在电气回路中，连于一点上的所有导线（包括接触连接的可折线段）须标以相同的回路标号。

3）电气设备的触点、线圈、电阻、电容等元件所间隔的线段

看为不同的线段，一般给予不同的标号。

对于在接线图中不经过端子而在屏内直接连接的回路，可不标号。
